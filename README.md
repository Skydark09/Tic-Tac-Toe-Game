# 🎮Tic-Tac-Toe Web Application

A fully functional Tic-Tac-Toe game built using HTML, CSS, and JavaScript.  
The application supports both multiplayer mode and a smart AI-powered computer opponent.

This project demonstrates front-end development skills, DOM manipulation, and logical problem-solving through AI-based decision-making.

------------------------------------------------------------

## **Key Highlights**

- Two-Player Mode (Local multiplayer)
- Player vs Computer Mode
- Smart AI Opponent:
    • Detects and plays winning moves
    • Blocks opponent’s winning opportunities
    • Applies strategic move selection logic
- Automatic win and draw detection
- Reset and replay functionality
- Clean and intuitive user interface

------------------------------------------------------------
## **AI Logic Overview**

The computer opponent follows a structured decision process:

1. Check for a winning move
2. Block the opponent if they are about to win
3. Select the best available move

This ensures the AI plays competitively rather than randomly.
