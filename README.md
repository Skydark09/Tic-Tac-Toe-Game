# Tic-Tac-Toe Game

A simple **Tic-Tac-Toe** game built with **HTML, CSS, and JavaScript**.

## Features
- 2-player mode (working)
- Single-player mode vs computer (in progress)
- Interactive UI with win/draw detection
- Reset and new game functionality

## Future Improvements
- Complete computer AI for single-player mode
- Enhance UI for mobile responsiveness
- Add difficulty levels for AI

## How to Play
1. Open `ind.html` in a browser.
2. Select a mode: 2 Players or Computer vs User.
3. Take turns to place X or O.
4. The game detects wins, draws, and allows resets.

## Technologies
- HTML, CSS, JavaScript
